🧠 算法题：两数之和（Two Sum）

🎯 题目描述：

给定一个整数数组 nums 和一个目标值 target，请你在数组中找出两个加起来等于目标值的数字，并返回它们的数组下标。

你可以假设每种输入只会对应一个答案。但数组中同一个元素不能使用两遍。

💡 示例：

✅ 解题要求：
• 时间复杂度尽量做到 O(n)
• 返回一组下标即可，无需考虑顺序
• 提供 Swift 或 Objective-C 实现

✨ 提示：

你可以用哈希表记录访问过的数字及其下标，在遍历时直接判断 target - 当前值 是否存在于 map 中。

🚀 拓展思考（建议尝试）：
1. 如果要求返回所有可能的两数组合（即可能有多个答案），你会怎么做？
2. 如果输入是一个有序数组，你是否可以用双指针优化？
